# Light & Dark Mode Toggle

- This project showcases a simple and user-friendly Light & Dark Mode system.
Users can switch between light and dark themes instantly, and the website automatically remembers their choice even after refreshing or reopening the page.

## Light Mode

- A bright, clean theme designed for daytime reading and clarity.

## Dark Mode

- A darker theme that reduces eye strain and is ideal for night use.

## Purpose of the Project

- The main purpose of this project is to demonstrate:

- How to switch between light and dark themes dynamically

- How to update UI elements such as text, icons, and backgrounds

- How to store and load theme preferences

- How to improve user experience with personalized appearance settings
